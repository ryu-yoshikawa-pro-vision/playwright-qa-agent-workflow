# 00_request

このファイルはサンプルです。実案件の依頼正本ではありません。

## 依頼内容

Mini SaaS のログイン機能について、Playwright CLI による画面探索結果をもとに、テスト計画、テスト設計、生成前レビュー、Playwright Test 生成例、coverage 台帳までの流れを示す。

## 対象範囲

- ログイン画面
- ログイン成功後のダッシュボード画面
- 入力バリデーション
- 認証失敗時のエラー表示

## 対象外

- パスワード再設定
- アカウント作成
- 多要素認証
- 権限別メニュー
- 本物のメール送信
- 実在サービスへのアクセス

## 成功条件

- ログイン機能の仕様とテスト観点が分かる。
- どの証跡を根拠にしたかが分かる。
- 生成前レビューで `PASS` と判断できる材料がそろう。
- 生成される Playwright Test の形が分かる。
