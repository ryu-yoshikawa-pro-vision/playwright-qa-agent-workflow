# ログイン機能

このファイルはサンプルです。実際の機能仕様ではありません。

## 機能 ID

FTR-001

## 概要

有効な一般ユーザーが、メールアドレスとパスワードでログインし、ダッシュボードへ遷移する。

## 対象画面

- SCR-001 ログイン画面
- SCR-002 ダッシュボード画面

## 主なルール

- メールアドレスは必須。
- パスワードは必須。
- メールアドレス形式が不正な場合はエラーを表示する。
- 認証情報が誤っている場合は認証失敗エラーを表示する。
- 認証に成功した場合はダッシュボードへ遷移する。

## 対象外

- アカウント作成
- パスワード再設定
- 多要素認証
- アカウントロック
