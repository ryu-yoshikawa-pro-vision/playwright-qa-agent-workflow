# ログインフロー

このファイルはサンプルです。実際のフロー仕様ではありません。

## 正常フロー

1. ユーザーが `/login` を表示する。
2. メールアドレスを入力する。
3. パスワードを入力する。
4. ログインボタンを押す。
5. 認証に成功する。
6. `/dashboard` に遷移する。
7. `ダッシュボード` 見出しが表示される。

## 異常フロー

### 必須入力エラー

1. ユーザーが `/login` を表示する。
2. メールアドレスまたはパスワードを空欄にする。
3. ログインボタンを押す。
4. 対象項目の必須入力エラーが表示される。

### メール形式エラー

1. ユーザーが `/login` を表示する。
2. メール形式ではない文字列を入力する。
3. ログインボタンを押す。
4. メール形式エラーが表示される。

### 認証失敗エラー

1. ユーザーが `/login` を表示する。
2. 存在しない認証情報、または誤ったパスワードを入力する。
3. ログインボタンを押す。
4. 認証失敗エラーが表示される。
